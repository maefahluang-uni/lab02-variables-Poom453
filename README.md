### Instruction
นี่คือสูตรสำหรับการแปลง:
```
C = (F - 32) * 5/9
```
โดยที่ F คืออุณหภูมิในหน่วยฟาเรนไฮต์และ C คืออุณหภูมิในหน่วยเซลเซียส
